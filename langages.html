Display : block

BLOCK
BLOCK
BLOCK [Padding only]
BLOCK [Margin only]
BLOCK [Padding + Margin]

A 'block' element is placed on a new line within its container, and stretches out to occupy the whole space of its container.

>>> Padding is fully applied.

>>> Margin is fully applied.

Display : inline

INLINE
INLINE
INLINE [Padding only]
INLINE [Margin only]
INLINE [Padding + Margin]

An 'inline' element is placed on the current line of its container, and only occupy the necessary space for them.

>>> Padding is applied but does not expand the container !

>>> Margin is applied only on right and left !

Display : inline-block

INLINE-BLOCK
INLINE-BLOCK
INLINE-BLOCK [Padding only]
INLINE-BLOCK [Margin only]
INLINE-BLOCK [Padding + Margin]

An 'inline-block' element is placed on the current line of its container, and only occupy the necessary space for them.

>>> Padding is applied and it does expand the container.

>>> Margin is fully applied and also expand the container.
