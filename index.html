Qu'est ce que le projet 'TSP' ?

The Synthetic Project est une base de données personnelle en construction. Cette base contient des fiches concepts pour le développement Web.

L'objectif initial de cette page est de constituer un exercice 'use-case' accompagnant l'apprentissage auto-didacte des langages pour le développement Full-Stack.
