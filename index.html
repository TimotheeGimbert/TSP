The Synthetic Project

Le Projet Synthetic, c'est quoi ?

C'est un projet Web personnel prenant la forme d'une base de données relatives à la programmation informatique.

Constituée de fiches Synthetic, cette base essayera de faire la synthèse des concepts clés du développement Web.

HTML

Architecture web

Structure du Document Object Model

CSS

Mise en page et style

Positionner et styliser les éléments

JavaScript

Dynamisation de l'environnement

Etablir des interactions avec le DOM

Bootstrap

Framework HTML

Pour des mises en page responsive

Sass

Préprocesseur CSS

Structure l'utilisation du CSS

React

Framework JavaScript

Interagir avec le DOM
