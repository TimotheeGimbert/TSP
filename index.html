The Synthetic Way c'est ...

... une base de données naissante visant à synthétiser des notions clés du développement Web.

C'est un peu un site web sur ... ' Comment Faire Un Site Web ? ' ... !

On trouve à ce sujet déjà de très bonnes ressources, nombreuses sur le web. The Synthetic Way vise à condenser les informations clés afin de partager une vision d'ensemble.

Tout cela n'est encore qu'un exercice personnel, peut-être éphemère, qui fait parti de mon auto-formation de développeur justement, donc ce site se construit doucement !
