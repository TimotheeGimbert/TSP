The Synthetic Project

Le Projet Synthetic c'est ...

... le partage d'une méthode de développement Web Front-End pour les développeurs en formation.

Constitué de fiches 'synthetic', le projet commencera par résumer des concepts clés.

Dans un second temps, il partagera une méthode responsive et une approche globale pour organiser son code.
